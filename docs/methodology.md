# Core Trading Concepts

## Liquidity Sweeps
Price wicks beyond obvious levels to trigger stops.

## Reclaim Logic
A level is only valid once price closes back through it.

## Fibonacci CC Levels
Beyond 0.618:
- 0.66 (CC)
- 0.71 (CC 3.0)

## Market State Awareness
Indicators assume:
- Range
- Correction
- Non-trending environments
